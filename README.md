# Portfolio académique — Séraphin Eyala

Portfolio académique du **BUT 3 Informatique** (IUT de Villetaneuse · USPN) : démarche réflexive
de valorisation et d'auto-évaluation des compétences.

> 🎨 Squelette de design (React + Vite, style « coloré & créatif »). Le contenu réel se remplit
> dans `src/data/content.js` — tous les champs marqués `TODO`.

## Structure du site

| Section | Contenu |
|---|---|
| **Hero** | Identité + accroche |
| **Introduction** | Logique du parcours, orientation, ambitions, poursuite d'études |
| **Les 6 compétences** | Réaliser · Optimiser · Administrer · Gérer · Conduire · Collaborer |
| **Bilan des SAÉ** | Mémos + tableaux d'auto-évaluation |
| **Stage BUT 3** | Missions + recul critique |
| **Conclusion** | Points forts, vigilance, acquis pour la suite |

## Démarrer

```bash
npm install
npm run dev      # serveur local
npm run build    # build de production -> dist/
```

## Remplir le contenu

Tout le texte vit dans **`src/data/content.js`**. Remplace chaque `TODO` par tes vraies infos —
le design s'adapte automatiquement.

## Déploiement

Le workflow `.github/workflows/deploy.yml` publie automatiquement sur **GitHub Pages**
à chaque push sur `main` (activer Pages → *Source : GitHub Actions* dans les réglages du repo).
